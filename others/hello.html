防抖与节流

# 防抖：当事件持续触发时不会执行，等最后一次触发完毕一段时间才执行一次。

使用场景：如输入框搜索，不应该一直请求服务器，当用户停止输入的一段时间内再触发

两个关键点：1.一段时间后才执行一次；

​ 2.事件持续触发不执行。

// 满足第一个条件
function debounce(fnc, wait){
    return function(){
        setTimeout(()=> {
        	fnc.apply(this.arguments)
    	},wait)
    }
}

// 满足第二个条件
function debounce(fnc, wait){
    let timer;
    return function(){
        clearTimeout(timer)
        timer = setTimeout(()=> {
        	fnc.apply(this.arguments)
    	},wait)
    }
}

# 节流：一段时间内执行一次

使用场景：浏览器mousemove事件，不能一直触发，应当一段事件内触发一次

两个关键点：1.到一定时间内执行

​ 2.事件持续触发不会多次执行

function throttle(fnc, wait){
    return function(){
        let time
        if(time) return
        time = true
        setTimeout(()=> {
            fnc.apply(this, arguments)
            time = false
        }, wait)
    }
}
