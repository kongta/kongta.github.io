    # 遍历

    # 遍历的方法

    1. for循环
        let fruits = ['apple', 'banana', 'pear', 'mango']
    for (let i = 0; i < fruits.length; i++) {
        console.log(fruits[i])
    }
    // 'apple', 'banana', 'pear', 'mango'

    这种写法对于计算机来说可读性是很高的，但是对我们来说可读性不是很好,获取当前元素的值只能靠下标

    1. forEach
        let fruits = ['apple', 'banana', 'pear', 'mango']
    fruits.forEach((item, i) => {
        console.log(item, i)
    })
    // 'apple' 0, 'banana' 1, 'pear' 2, 'mango' 3

    可以同时拿到属性跟下标

    let fruits = ['apple', 'banana', 'pear', 'mango']
    fruits.forEach((item) => {
        if (item === 'pear') {
            break;
        }
        console.log(item)
    })
    fruits.forEach((item) => {
        if (item === 'pear') {
            continue;
        }
        console.log(item)
    })

    这种循环有个缺点，不能终止或者跳过循环，当数组很大的时候可能存在性能问题，以上两种写法都会报错

    1. for in 语法简单
        let fruits = ['apple', 'banana', 'pear', 'mango']
    for (let index in fruits) {
        console.log(fruits[index])
    }
    // 'apple', 'banana', 'pear', 'mango'

    这种方法通过循环拿到的是属性名，而不是属性值

        let fruits = ['apple', 'banana', 'pear', 'mango']
    fruits.desc = 'delicious'
    for (let index in fruits) {
        console.log(fruits[index])
    }
    // 'apple', 'banana', 'pear', 'mango', 'delicious'

    需要注意的是for in 遍历的是可枚举属性

    1. for of 遍历可迭代对象（数组、Map、Set、类数组对象，如 arguments 对象、DOM NodeList 对象、字符串） 遍历数组（得到是属性值、可以终止循环、不会遍历非数字属性）
    
    let fruits = ['apple', 'banana', 'pear', 'mango']
    for(let [index,item] of fruits.entries()) {
        console.log(item)
    }
    // 'apple', 'banana', 'pear', 'mango',

    let fruits = ['apple', 'banana', 'pear', 'mango']
    for (let item of fruits) {
        if (item === 'pear') {
            break;
        }
        console.log(item)
    }
    // 'apple', 'banana'

    遍历字符串

        let fruits = 'fruits'
    for (let item of fruits) {
        console.log(item)
    }
    // 'f','r','u','i','t','s'

    还支持nodeList....

    # 遍历遇到的问题

    当我们遍历需要操作原数组时候 我们的思路都是循环判断然后删除

        let fruits = ['apple', 'banana', 'pear', 'mango','banana','banana','banana','banana']
    for (let i = 0; i < fruits.length; i++) {
        if (fruits[i] === 'banana' ) {
            fruits.splice(i,1)
        }
    }
    console.log(fruits) // ["apple", "pear", "mango", "banana", "banana"]

    然而结果却跟我们想象中的不同，因为改变的时候数组的下标也跟着改变了

    # 解决方法

    1. 判断的时候修改下标
        let fruits = ['apple', 'banana', 'pear', 'mango','banana','banana','banana','banana']
    for (let i = 0; i < fruits.length; i++) {
        if (fruits[i] === 'banana' ) {
            fruits.splice(i,1)
            i--
        }
    }
    console.log(fruits) // ["apple", "pear", "mango"]
    1. 从后往前循环
        let fruits = ['apple', 'banana', 'pear', 'mango','banana','banana','banana','banana']
    for (let i = fruits.length; i >= 0; i--) {
        if (fruits[i] === 'banana' ) {
            fruits.splice(i,1)
        }
    }
    console.log(fruits) // ["apple", "pear", "mango"]
